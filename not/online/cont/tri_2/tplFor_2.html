PHP 安装

安装前所需要下载的

下载 PHP

从此处下载免费的 PHP：http://www.php.net/downloads.php

下载 MySQL 数据库

从此处下载免费的 MySQL：http://www.mysql.com/downloads/

下载 Apache 服务器

从此处下载免费的 Apache：http://httpd.apache.org/download.cgi

安装 Apache

双击安装包，疯狂下一步。httpd.conf在{Apache根目录}/conf/下

安装 MySQL

双击安装包，疯狂下一步。记得设置root用户的密码。

安装 PHP

PHP安装包解压到某个你喜欢的文件夹就行了，比如 php-5.2.13-Win32.zip 文件解压到 C:\php 下

将 PHP 以模块的方式加载到Apache中

把下面三行代码粘贴到httpd.conf文件里面

LoadModule php5_module "C:\php\php5apache2_2.dll" #加载PHP模块

AddType application/x-httpd-php .php .phtml #PHP模块解析哪种后缀文件

PHPIniDir "C:\php" #指定PHP配置文件的位置

我上面设置的 php目录 是 C:\php，改成你自己的就行了。

建立PHP的配置文件

直接把PHP解压缩目录C:\php下面的php.ini-recommended文件改名为php.ini,然后复制一份php.ini到C:\Windows下

检查一下PHP是否安装成功

首先重启Apache服务器，怎么重启，屏幕右下方那个很难看的图标就是了。

Apache服务器存放网页的文件夹{Apache根目录}\htdocs下,使用文本编辑器创建一个名为test.php的文件,写入
<?php phpinfo();?>

然后使用浏览器打开http://localhost/test.php页面。出现一个PHP测试页说明PHP安装成功。

配置 PHP 的 MySQL 扩展接口

在PHP配置文件中开启MySQL功能。使用文本编辑器打开PHP的配置文件C:\lamp\php5\php.ini。

插入下面三项就可以开启mysql和mysqli扩展接口。也可以在PHP配置文件中找到下面三行，将前面用于注释的分号去掉。

extension_dir="c:/lamp/php5/ext" #指定PHP的扩展接口目录，目录分隔符是”/”

extension=php_mysql.dll #开启MySQL扩展接口

extension=php_mysqli.dll #开启MySQLi扩展接口

重启Apache
