PHP 语法

您无法在浏览器中通过查看源文档的方式来查看 PHP 的源代码 - 您只能看到 PHP 文件的输出，即纯粹的 HTML。这是因为在页面返回浏览器之前，脚本就已经在服务器执行了。

基本的 PHP 语法

PHP 的代码块以 <?php 开始，以 ?> 结束。您可以把 PHP 的代码块放置在页面文档中的任何位置。

当然，在支持简写的服务器上，您可以使用 <? 和 ?> 来开始和结束脚本块。

为了达到最好的兼容性，亿柿网推荐你使用标准形式 (<?php)，而不是简写形式。

<?php ?>

PHP 文件通常会包含 HTML 标签，就像一个 HTML 文件，以及一些 PHP 脚本代码。

下面是一段简单的 PHP 脚本，它可以向浏览器输出文本 "Hello World"：

<html>
<body>

<?php
echo "Hello World";
?>

</body>
</html>

PHP 中的每个代码行都必须以分号结束。分号是一种分隔符，用于把指令集区分开来。

PHP 有两种基础指令来输入文本：echo 和 print。在上面的例子中，我们使用了 echo 语句来输出文本 "Hello World"。

在性能优化方面，我们推荐使用 echo 语句来输出文本。因为print输出后会返回true或false，即输出是否成功，在大量输出的情况echo会比print要快。

PHP 中的注释

在 PHP 中，我们使用 // 来编写单行注释，或者使用 /* 和 */ 来编写大的注释块。

也可以使用 # 来进行单行注释

<html>
<body>
<?php
//我是注释

/*
我是
注释
块
*/
?>
</body>
</html>
