PHP 简介

PHP 文件可包含文本、HTML 标签以及脚本。PHP 文件中的脚本可在服务器上执行。

您应当具备的基础知识

在继续学习之前，你至少需要对下面的知识有基本的了解：

  • C/C++
  • HTML / XHTML

什么是 PHP？

  • PHP 指 PHP:超文本预处理器（注：PHP: Hypertext Preprocessor）
  • PHP 是一种服务器端的脚本语言，类似 ASP
  • PHP 脚本在服务器上执行
  • PHP 支持很多数据库（MySQL、Informix、Oracle、Sybase、Solid、PostgreSQL、Generic ODBC 等等）
  • PHP 是一个开源的软件（open source software，OSS）
  • PHP 可免费下载使用

什么是 PHP 文件？

  • PHP 文件可包含文本、HTML 标签以及脚本
  • PHP 文件向浏览器返回纯粹的 HTML
  • PHP 文件的文件后缀是 ".php" 或 ".phtml"

什么是 MySQL？

  • MySQL 是一种数据库服务器
  • MySQL 支持标准的 SQL
  • MySQL 可在许多平台上编译
  • MySQL 可免费下载使用

PHP + MySQL

PHP 与 MySQL 的组合是跨平台的（意思是您可以在 Windows (WMAP) 环境进行开发，而在 Unix (LAMP) 平台上提供服务）

为什么要使用 PHP？

  • PHP 可在不同的平台上运行（Windows、Linux、Unix）
  • PHP 与目前几乎所有的正在被使用的服务器相兼容（Apache、IIS 等）
  • PHP 可从官方的 PHP 资源免费下载：www.php.net
  • PHP 容易上手，并可高效地运行在服务器端
